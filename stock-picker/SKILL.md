# stock-picker - 选股专家

**理论来源**：《彼得·林奇的成功投资》

**功能**：基于林奇 13 条原则选股，PEG 估值，生活选股法

**触发词**：
- "如何选股"
- "这家公司值得投资吗"
- "PEG 是多少"
- "林奇选股法"

---

## 输入参数

```json
{
  "company_name": "公司名称",
  "business_description": "业务描述",
  "pe_ratio": "市盈率",
  "growth_rate": "盈利增长率",
  "observation": "生活中的观察（可选）"
}
```

### 3. PEG 计算和评估

```
PEG = 市盈率 (PE) / 盈利增长率 (G)

当前 PEG = X / Y = Z

评估：
- <0.5：极佳（强烈买入）
- 0.5-1.0：良好（买入）
- 1.0-1.5：合理（持有）
- 1.5-2.0：偏高（谨慎）
- >2.0：过高（避免）
```

### 4. 生活选股法

**彼得·林奇建议**：
- 从生活中发现好公司
- 你了解的行业优先

**你的观察**：
- 记录你在生活中发现的好产品/服务
- 你身边的人在用什么

## 使用示例

**输入**：
```
公司：某消费品公司
业务：生产日常消费品
PE：25
增长率：30%
观察：身边人都在用，复购率高
```

**输出**：
```
【公司类型】
稳定增长型/快速增长型

【林奇 13 条原则检查】
符合 X/13 条（已知信息）

【PEG 计算】
PEG = 25 / 30 = 0.83
评估：良好（0.5-1.0 区间）

【生活选股法】
✅ 身边人都在用
✅ 复购率高
✅ 你了解这个行业

【投资建议】
评级：推荐
理由：
1. PEG 合理（0.83）
2. 增长率良好（30%）
3. 消费品，需求稳定
4. 生活观察验证

建议：
- 进一步确认：机构持股、内部人交易、财务健康度
- 买入策略：分批买入
- 持有期限：3-5 年
```
