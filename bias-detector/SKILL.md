# bias-detector - 认知偏差检测器

**理论来源**：《思考，快与慢》- 丹尼尔·卡尼曼

**功能**：识别投资决策中的 8 种核心认知偏差，提供改善建议

**触发词**：
- "我是否有认知偏差"
- "检查这个决策"
- "为什么我总是亏钱"
- "决策检查"

---

## 输入参数

```json
{
  "decision_context": "决策背景描述",
  "your_thinking": "你的思考过程",
  "emotion_state": "当前情绪状态（兴奋/焦虑/恐惧/贪婪）",
  "position_status": "持仓状态（空仓/持仓/加仓/减仓）"
}
```

## 8 种核心偏差详解

### 1. 损失厌恶 (Loss Aversion)

**表现**：
- 亏损的痛苦 > 盈利的快乐（约 2 倍）
- 不愿止损，希望"回本"
- 过早卖出盈利股票

**检查问题**：
- 你是否死守亏损的股票？
- 你是否过早卖出盈利的股票？
- 你是否因为怕亏而错过机会？

**改善建议**：
- 忘记买入价，关注当前价值
- 设定明确的止损规则
- 记录决策理由，定期复盘

### 2. 锚定效应 (Anchoring)

**表现**：
- 被买入价/目标价锚定
- 过度依赖第一个信息
- 忽视新信息

**检查问题**：
- 你是否总说"等回本就卖"？
- 你是否过度关注买入价？
- 新信息出现时，你调整观点了吗？

**改善建议**：
- 定期忘记买入价
- 基于当前信息重新评估
- 多角度获取信息

### 3. 确认偏误 (Confirmation Bias)

**表现**：
- 只找支持自己观点的信息
- 忽视相反证据
- 只关注利好消息

**检查问题**：
- 你是否只看好评/利空？
- 你是否搜索反对意见？
- 你能说出 3 个看空理由吗？

**改善建议**：
- 主动寻找反面观点
- 列出反对理由清单
- 找持相反观点的人讨论

### 4. 处置效应 (Disposition Effect)

**表现**：
- 过早卖出盈利
- 死守亏损
- 卖出赢家，留下输家

**检查问题**：
- 你的盈利股持有时间 vs 亏损股？
- 你是否"落袋为安"？
- 你是否希望亏损股"回本"？

**改善建议**：
- 基于基本面决策，而非盈亏
- 设定统一的持有标准
- 定期审视持仓

### 5. 从众心理 (Herding)

**表现**：
- 跟随大众买卖
- 害怕错过 (FOMO)
- 忽视独立判断

**检查问题**：
- 你买入是因为大家都买吗？
- 你是否害怕错过机会？
- 你能独立论证买入理由吗？

**改善建议**：
- 远离噪音（股吧、媒体）
- 写投资笔记，记录理由
- 逆向思考训练

### 6. 过度自信 (Overconfidence)

**表现**：
- 高估自己的能力
- 过度交易
- 忽视风险

**检查问题**：
- 你认为自己比市场聪明吗？
- 你的交易频率如何？
- 你记录过自己的准确率吗？

**改善建议**：
- 记录每次决策和理由
- 定期计算准确率
- 承认市场不可预测

### 7. 代表性偏差 (Representativeness)

**表现**：
- 以偏概全
- 过度推断
- 忽视基础概率

**检查问题**：
- 你是否因为 3 次成功就认为策略有效？
- 你是否忽视长期概率？
- 小样本能代表整体吗？

**改善建议**：
- 关注大样本数据
- 了解基础概率
- 避免过度推断

### 8. 可得性偏差 (Availability Heuristic)

**表现**：
- 依据容易想起的信息决策
- 过度重视近期事件
- 忽视历史数据

**检查问题**：
- 你是否过度重视最近的消息？
- 你能想起多久以前的案例？
- 你查阅历史数据了吗？

**改善建议**：
- 查阅长期历史数据
- 建立投资数据库
- 避免仅凭记忆决策

## 使用示例

**输入**：
```
决策背景：我持有的股票跌了 20%，要不要止损？
思考过程：我觉得会反弹，再等等
情绪状态：焦虑 + 不甘
持仓状态：持仓，亏损 20%
```

**输出**：
```
【认知偏差检查】

| 偏差类型 | 是否存在 | 证据 | 影响 |
|---------|---------|------|------|
| 损失厌恶 | ✅ | 不愿止损，希望回本 | 可能扩大亏损 |
| 锚定效应 | ✅ | 被买入价锚定 | 无法客观评估 |
| 确认偏误 | ✅ | 只找利好信息 | 忽视风险 |
| 处置效应 | ✅ | 死守亏损股 | 资金效率低 |

【偏差影响】
损失厌恶导致不愿承认错误，可能扩大亏损
锚定效应让你关注买入价而非公司价值
确认偏误让你无法客观评估风险

【改善建议】
立即行动：
1. 忘记买入价，问："如果现在空仓，还会买入吗？"
2. 重新评估公司基本面
3. 设定明确的止损规则

长期改善：
1. 记录每次决策理由
2. 定期复盘，计算准确率
3. 建立投资检查清单

【决策检查清单】
□ 公司基本面是否变化？
□ 当初买入理由是否还成立？
□ 是否有更好的投资机会？
□ 止损是否符合投资计划？
□ 决策是否受情绪影响？
```
