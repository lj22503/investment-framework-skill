# industry-analyst - 行业分析师

**理论来源**：《如何快速了解一个行业》- 肖璟

**功能**：快速分析行业生命周期、竞争格局、投资价值

**触发词**：
- "分析 XX 行业"
- "这个行业值得投资吗"
- "行业研究框架"

---

## 输入参数

```json
{
  "industry_name": "行业名称",
  "industry_description": "行业描述",
  "market_size": "市场规模（可选）",
  "growth_rate": "增长率（可选）",
  "key_players": "主要玩家（可选）"
}
```
