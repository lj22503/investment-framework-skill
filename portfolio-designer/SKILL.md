# portfolio-designer - 组合设计师

**理论来源**：《机构投资者的创新之路》- 大卫·史文森（耶鲁模式）

**功能**：基于耶鲁模式设计资产配置方案

**触发词**：
- "设计资产配置方案"
- "耶鲁模式怎么配置"
- "我有 X 万如何配置"

---

## 输入参数

```json
{
  "age": "年龄",
  "total_assets": "总资产",
  "risk_tolerance": "风险偏好（保守/平衡/积极）",
  "investment_goal": "投资目标",
  "time_horizon": "投资期限"
}
```
