# 投资框架 Skill 包

## 概述

基于 5 本经典投资书籍的实战工具箱：
- 《聪明的投资者》格雷厄姆 - 安全边际、市场先生
- 《证券分析》格雷厄姆 & 多德 - 内在价值、财务分析
- 《巴菲特致股东的信》巴菲特 - 护城河、能力圈
- 《穷查理宝典》芒格 - 多元思维、逆向思考
- 《漫步华尔街》马尔基尔 - 指数基金、资产配置

## 核心技能

### 1. value-analyzer（价值分析师）
**理论来源**：《聪明的投资者》
**功能**：基于格雷厄姆原则分析股票价值
**输入**：股票代码/公司名称、财务数据
**输出**：防御型/积极型评估、安全边际计算、投资建议

### 2. moat-evaluator（护城河评估师）
**理论来源**：《巴菲特致股东的信》
**功能**：评估企业护城河类型和强度
**输入**：公司描述、商业模式、竞争格局
**输出**：护城河类型、强度评分、持续性评估

### 3. asset-allocator（资产配置师）
**理论来源**：《漫步华尔街》
**功能**：根据生命周期设计资产配置方案
**输入**：年龄、风险偏好、投资目标、资产规模
**输出**：资产配置比例、再平衡策略、基金推荐

### 4. decision-checklist（决策清单）
**理论来源**：《穷查理宝典》
**功能**：基于多元思维和认知偏差检查投资决策
**输入**：投资想法、决策理由
**输出**：认知偏差检查、逆向思考、决策建议

### 5. intrinsic-value-calculator（内在价值计算器）
**理论来源**：《证券分析》
**功能**：计算企业内在价值和安全边际
**输入**：财务数据（资产、盈利、现金流等）
**输出**：内在价值、安全边际、买入/卖出建议

### 6. future-forecaster（未来预测师）🆕
**理论来源**：凯文·凯利《必然》《失控》
**功能**：基于 KK 三思维模型和四层变化规律识别长期趋势
**输入**：行业观察、技术现象、新词汇/新概念
**输出**：趋势分类、爆发时点预估、变化层级判定、投资建议

## 组合使用流程

### 个股投资流程
```
1. value-analyzer → 初步筛选（是否符合格雷厄姆标准）
2. moat-evaluator → 护城河评估（是否有持续竞争优势）
3. intrinsic-value-calculator → 价值计算（内在价值和安全边际）
4. decision-checklist → 决策检查（避免认知偏差）
```

### 趋势驱动型投资流程 🆕
```
1. future-forecaster → 识别长期趋势和方向
2. moat-evaluator → 评估相关公司护城河能否抵御变化
3. value-analyzer → 筛选符合趋势的价值标的
4. intrinsic-value-calculator → 计算合理价格
5. decision-checklist → 检查认知偏差和 FOMO
6. asset-allocator → 配置到组合中合适位置
```

### 趋势陷阱规避流程 🆕
```
1. future-forecaster (layer-analyzer) → 判断是趋势还是流行
2. 若为流行趋势 → 避免重大投资
3. 若为技术变革 → 进入深度分析流程
```

### 资产配置流程
```
1. asset-allocator → 设计配置方案
2. value-analyzer → 选择具体标的
3. decision-checklist → 最终决策检查
```

### 能力圈扩展流程 🆕
```
1. future-forecaster (mindset-checker) → 检查认知捆绑
2. 识别过时认知 → 主动学习新领域
3. 扩大能力圈 → 纳入投资框架分析
```

## 核心理论框架

### 格雷厄姆安全边际原则
- 用 4 毛钱买 1 块钱的东西
- 安全边际至少 30%
- 防御型投资者：股债 50:50
- 积极型投资者：深度研究，集中投资

### 巴菲特护城河理论
- 品牌优势
- 网络效应
- 转换成本
- 规模优势
- 特许经营权

### 芒格多元思维
- 学习重要学科的重要理论
- 逆向思考
- 识别认知偏差
- 能力圈原则

### 马尔基尔资产配置
- 指数基金定投
- 生命周期配置
- 长期持有
- 定期再平衡

## 输出模板

### 投资价值分析报告
```
【公司名称】[名称]
【评估类型】防御型/积极型
【安全边际】X%
【护城河】类型 + 强度
【内在价值】¥X
【当前价格】¥Y
【投资建议】买入/持有/卖出
```

### 资产配置方案
```
【年龄】X 岁
【风险偏好】保守/平衡/积极
【股票】X%（国内 X%，国际 X%）
【债券】X%
【现金】X%
【另类】X%
【再平衡】每年 X 月
```

## 注意事项

1. **能力圈原则** - 只投资你理解的公司
2. **安全边际** - 价格必须低于价值
3. **长期思维** - 持有期至少 5 年
4. **避免杠杆** - 不用借来的钱投资
5. **持续学习** - 扩大能力圈
