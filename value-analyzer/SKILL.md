# value-analyzer - 价值分析师

## 理论来源
《聪明的投资者》- 本杰明·格雷厄姆 (1949)

## 核心功能
基于格雷厄姆原则分析股票价值，判断是否符合投资标准

## 输入参数
```json
{
  "company_name": "公司名称",
  "stock_price": "当前股价",
  "financial_data": {
    "market_cap": "市值",
    "pe_ratio": "市盈率",
    "pb_ratio": "市净率",
    "debt_ratio": "负债率",
    "roe": "净资产收益率",
    "dividend_yield": "股息率"
  },
  "earnings_history": "盈利历史（连续 10 年是否盈利）",
  "dividend_history": "分红历史（连续 20 年是否分红）",
  "investor_type": "防御型/积极型"
}
```

## 输出结构

### 1. 投资者类型评估

**防御型投资者标准**：
| 标准 | 要求 | 是否符合 |
|------|------|---------|
| 公司规模 | 行业前 30% | ✅/❌ |
| 财务状况 | 负债率<50% | ✅/❌ |
| 盈利稳定 | 连续 10 年盈利 | ✅/❌ |
| 分红记录 | 连续 20 年分红 | ✅/❌ |
| 盈利增长 | 10 年增长>33% | ✅/❌ |
| 股价资产比 | P/B < 1.5 | ✅/❌ |
| 市盈率 | P/E < 15 | ✅/❌ |

**积极型投资者标准**：
- 财务稳健（负债率<50%）
- 当前盈利为正
- 股价低于内在价值（安全边际>30%）

### 2. 安全边际计算

```
内在价值估算：
- 资产价值 = 流动资产 - 总负债
- 盈利价值 = 正常化盈利 × 10（合理 PE）
- 内在价值 = (资产价值 + 盈利价值) / 2

安全边际 = (内在价值 - 当前价格) / 内在价值 × 100%
```

### 3. 投资建议

| 安全边际 | 建议 | 说明 |
|---------|------|------|
| >50% | 强烈买入 | 极佳投资机会 |
| 30-50% | 买入 | 符合格雷厄姆标准 |
| 10-30% | 观察 | 价格接近价值 |
| <10% | 不买入 | 安全边际不足 |

### 4. 风险提示

- [ ] 行业风险
- [ ] 财务风险
- [ ] 市场风险
- [ ] 管理层风险

## 分析框架

### 格雷厄姆 7 大标准（防御型）

1. **适当的公司规模**
   - 大型企业（行业前 30%）
   - 小型企业波动大，不适合防御型

2. **强劲的财务状况**
   - 流动比率 > 2
   - 负债率 < 50%

3. **盈利的稳定性**
   - 连续 10 年盈利
   - 无亏损年份

4. **分红记录**
   - 连续 20 年分红
   - 证明盈利真实可靠

5. **盈利增长**
   - 10 年盈利增长>33%
   - 年均增长>3%

6. **适度的股价资产比**
   - P/B < 1.5
   - 避免支付过高溢价

7. **合理的市盈率**
   - P/E < 15
   - 避免为未来增长支付过高价格

### 安全边际原则

```
格雷厄姆的安全边际思想：

"用 4 毛钱买 1 块钱的东西"

核心逻辑：
1. 计算企业内在价值
2. 等待价格大幅低于价值
3. 差额就是安全边际
4. 安全边际保护你免受错误和意外

建议安全边际：
- 防御型：至少 30%
- 积极型：至少 50%
```

## 使用示例

**输入**：
```
公司名称：贵州茅台
当前股价：¥1700
财务数据：
- 市值：2.1 万亿
- 市盈率：30
- 市净率：8
- 负债率：10%
- ROE：30%
- 股息率：1.5%
盈利历史：连续 10 年盈利 ✅
分红历史：连续 20 年分红 ✅
投资者类型：防御型
```

**输出**：
```
【投资者类型评估】

防御型标准检查：
| 标准 | 要求 | 是否符合 |
| 规模 | 行业前 30% | ✅ |
| 财务 | 负债率<50% | ✅ (10%) |
| 盈利 | 连续 10 年 | ✅ |
| 分红 | 连续 20 年 | ✅ |
| 增长 | 10 年>33% | ✅ |
| P/B | <1.5 | ❌ (8) |
| P/E | <15 | ❌ (30) |

符合 5/7 标准

【安全边际计算】

由于 P/B 和 P/E 过高，无法计算安全边际
当前价格远高于格雷厄姆标准

【投资建议】

建议：不买入
理由：
1. 市盈率 30 远超格雷厄姆标准（<15）
2. 市净率 8 远超格雷厄姆标准（<1.5）
3. 安全边际为负

【风险提示】

- 估值风险：当前估值处于历史高位
- 市场风险：消费股整体估值偏高
- 建议：等待更好价格或选择其他标的
```
