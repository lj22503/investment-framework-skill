# intrinsic-value-calculator - 内在价值计算器

## 理论来源
《证券分析》- 格雷厄姆 & 多德 (1934)

## 核心功能
基于财务数据计算企业内在价值和安全边际

## 输入参数
```json
{
  "company_name": "公司名称",
  "financial_data": {
    "current_assets": "流动资产",
    "total_liabilities": "总负债",
    "current_price": "当前股价",
    "shares_outstanding": "总股本",
    "normalized_earnings": "正常化盈利（近 3 年平均）",
    "free_cash_flow": "自由现金流",
    "book_value": "净资产",
    "tangible_assets": "有形资产"
  },
  "growth_rate": "预期增长率（%）",
  "risk_free_rate": "无风险利率（%）",
  "industry_pe": "行业平均市盈率"
}
```

## 输出结构

### 1. 多种估值方法

**方法 1：格雷厄姆公式**
```
内在价值 = EPS × (8.5 + 2g)
其中：
- EPS = 每股收益
- 8.5 = 零增长公司的合理 PE
- g = 预期增长率（%）

修正公式（考虑利率）：
内在价值 = EPS × (8.5 + 2g) × 4.4 / Y
其中 Y = 当前 AAA 企业债利率
```

**方法 2：资产价值法**
```
流动资产价值 = 流动资产 - 总负债
每股流动资产价值 = 流动资产价值 / 总股本

内在价值 = 流动资产价值 × 折扣率（通常 2/3）
```

**方法 3：盈利价值法**
```
盈利价值 = 正常化盈利 × 合理 PE
合理 PE = 行业平均 PE 或 10-15

每股内在价值 = 盈利价值 / 总股本
```

**方法 4：DCF 简化版**
```
内在价值 = FCF × (1+g) / (r-g)
其中：
- FCF = 自由现金流
- g = 永续增长率（通常 3-5%）
- r = 折现率（通常 10%）
```

### 3. 安全边际评估

**安全边际计算**：
```
安全边际 = (内在价值 - 当前价格) / 内在价值 × 100%
```

| 安全边际 | 评级 | 建议 |
|---------|------|------|
| >50% | 极佳 | 强烈买入 |
| 30-50% | 良好 | 买入 |
| 10-30% | 一般 | 观察 |
| 0-10% | 不足 | 不买入 |
| <0% | 高估 | 卖出/做空 |

## 分析框架

### 格雷厄姆估值原则

```
格雷厄姆的核心思想：

1. 用 4 毛钱买 1 块钱的东西
2. 安全边际是投资成功的核心
3. 价格是你付出的，价值是你得到的
4. 市场短期是投票机，长期是称重机
```

### 正常化盈利计算

```
正常化盈利 = 近 3-5 年盈利平均值

目的：
1. 消除周期性波动
2. 排除一次性损益
3. 反映真实盈利能力

调整项：
- 加回：一次性损失、资产减值
- 扣除：一次性收益、资产处置收益
```

### 合理 PE 选择

| 公司类型 | 合理 PE | 说明 |
|---------|--------|------|
| 稳定增长 | 15-20 | 消费、医药等 |
| 周期性强 | 8-12 | 钢铁、煤炭等 |
| 高增长 | 20-30 | 科技、新能源等 |
| 衰退行业 | 5-8 | 传统媒体等 |

### 安全边际原则

```
安全边际的作用：

1. 保护免受估值错误
2. 保护免受意外事件
3. 提供超额收益空间

格雷厄姆建议：
- 防御型投资者：安全边际>30%
- 积极型投资者：安全边际>50%
```

## 使用示例

**输入**：
```
公司：贵州茅台
财务数据：
- 流动资产：2000 亿
- 总负债：500 亿
- 当前股价：¥1700
- 总股本：12.5 亿股
- 正常化盈利：500 亿
- 自由现金流：450 亿
- 净资产：1800 亿
预期增长率：10%
无风险利率：3%
行业平均 PE：25
```

**输出**：
```
【多种估值方法】

方法 1：格雷厄姆公式
内在价值 = EPS × (8.5 + 2g)
EPS = 500 亿 / 12.5 亿 = ¥40
内在价值 = 40 × (8.5 + 2×10) = ¥1140

方法 2：资产价值法
流动资产价值 = 2000 - 500 = 1500 亿
每股流动资产价值 = 1500 / 12.5 = ¥120
内在价值 = 120 × 2/3 = ¥80

方法 3：盈利价值法
盈利价值 = 500 亿 × 25 = 12500 亿
每股内在价值 = 12500 / 12.5 = ¥1000

方法 4：DCF 简化版
内在价值 = 450 × (1+5%) / (10%-5%) = 9450 亿
每股内在价值 = 9450 / 12.5 = ¥756

【估值结果汇总】

| 方法 | 内在价值 | 当前价格 | 安全边际 |
| 格雷厄姆 | ¥1140 | ¥1700 | -33% |
| 资产价值 | ¥80 | ¥1700 | -95% |
| 盈利价值 | ¥1000 | ¥1700 | -41% |
| DCF | ¥756 | ¥1700 | -55% |
| 平均 | ¥744 | ¥1700 | -56% |

【安全边际评估】

安全边际 = -56%
评级：高估
建议：卖出/不买入

【敏感性分析】

不同增长率：
| 增长率 | 内在价值 | 安全边际 |
| 0% | ¥952 | -44% |
| 5% | ¥1045 | -39% |
| 10% | ¥1140 | -33% |

【投资建议】

综合评级：高估
建议仓位：0%

理由：
1. 所有估值方法显示高估
2. 安全边际为负（-56%）
3. 当前价格远高于内在价值
4. 不符合格雷厄姆安全边际原则

建议：
- 持有者：考虑减仓或卖出
- 未持有者：等待更好价格
- 理想买入价：<¥500（安全边际>30%）
```
