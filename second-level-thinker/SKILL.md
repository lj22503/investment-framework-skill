# second-level-thinker - 第二层思维者

**理论来源**：《投资最重要的事》- 霍华德·马克斯

**功能**：培养第二层思维能力，判断市场周期位置，逆向思考

**触发词**：
- "市场现在处于什么周期"
- "第二层思维怎么看"
- "市场共识是什么"
- "逆向思考"

---

## 输入参数

```json
{
  "market_situation": "当前市场情况描述",
  "consensus_view": "市场主流观点",
  "asset_class": "资产类别（股票/债券/商品等）",
  "sentiment_indicators": "情绪指标（可选）"
}
```

## 输出结构

### 1. 第一层思维分析（市场共识）

| 观点 | 逻辑 | 问题 |
|------|------|------|
| 市场共识 1 | ... | 过于简单？ |
| 市场共识 2 | ... | 忽视风险？ |

### 2. 第二层思维分析（深入思考）

| 问题 | 第一层回答 | 第二层回答 |
|------|-----------|-----------|
| 价格是否合理？ | 合理/不合理 | 相比什么？ |
| 共识是否正确？ | 是/否 | 何时会错？ |
| 风险在哪里？ | 显而易见的风险 | 隐藏的风险 |

### 3. 市场周期位置判断

| 指标 | 当前状态 | 周期信号 |
|------|---------|---------|
| 估值水平 | 高/中/低 | ⚠️/➡️/✅ |
| 市场情绪 | 乐观/中性/悲观 | ⚠️/➡️/✅ |
| 新闻热度 | 全是好消息/混合/全是坏消息 | ⚠️/➡️/✅ |
| 信贷条件 | 宽松/正常/紧缩 | ⚠️/➡️/✅ |

**综合判断**：扩张早期/扩张晚期/衰退早期/衰退晚期/复苏

### 4. 逆向思考建议

**如果市场是对的**：
- 应该做什么？

**如果市场是错的**：
- 错在哪里？
- 何时会被证明是错的？
- 如何从中获利？

## 分析框架

### 第二层思维核心问题

1. 价格是否已经反映了共识预期？
2. 什么情况下共识会是错的？
3. 我的观点与市场有何不同？
4. 我为什么认为自己是对的？
5. 如果我是错的，会发生什么？

### 市场周期指标

| 周期阶段 | 估值 | 情绪 | 新闻 | 信贷 | 建议 |
|---------|------|------|------|------|------|
| 扩张早期 | 低 | 悲观 | 坏消息 | 紧缩 | 买入 |
| 扩张晚期 | 高 | 乐观 | 好消息 | 宽松 | 谨慎 |
| 衰退早期 | 中高 | 转弱 | 混合 | 转紧 | 减仓 |
| 衰退晚期 | 低 | 极度悲观 | 坏消息 | 紧缩 | 布局 |

## 使用示例

**输入**：
```
市场情况：AI 科技股大涨，纳斯达克创新高
市场共识：AI 是未来，应该买入科技股
资产类别：股票（科技股）
```

**输出**：
```
【第一层思维】
- AI 是未来趋势 → 买入科技股
- 大家都看好 → 跟着买
- 创新高 → 强势继续

【第二层思维】
- AI 确实是未来，但当前价格是否已反映？
- 市场共识过于一致，是否存在泡沫？
- 哪些公司真正受益？哪些只是蹭热点？
- 如果我是卖方，会现在卖出还是买入？

【周期位置】
| 指标 | 当前 | 信号 |
| 估值 | 历史高位 | ⚠️ |
| 情绪 | 极度乐观 | ⚠️ |
| 新闻 | 全是 AI 好消息 | ⚠️ |
| 信贷 | 正常 | ➡️ |

综合判断：扩张晚期 → 接近顶部

【逆向思考】
如果市场是对的：
- AI 确实改变世界，龙头公司继续上涨

如果市场是错的：
- 大部分 AI 公司无法盈利
- 估值过高，需要时间消化
- 只有真正有技术/盈利的公司存活

【投资建议】
| 投资者 | 建议 | 理由 |
| 持有者 | 逐步减仓，锁定利润 | 估值过高 |
| 未持有者 | 等待回调，或选择被忽视的细分 | 避免追高 |
| 空仓者 | 不追高，关注其他被低估领域 | 风险收益比不佳 |
```

---

## 核心原则

1. **永远不要相信市场共识** - 共识往往已经反映在价格中
2. **寻找第二层机会** - 别人恐惧时贪婪，别人贪婪时恐惧
3. **保持独立思考** - 不随波逐流
4. **承认不确定性** - 市场可能长时间保持非理性
